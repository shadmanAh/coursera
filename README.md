**Installation Requirements**

- ruby v 3.0.2 +
- rails 7.0.2 +
- postgresql database
- yarn

## Installation the app

1. Create app

```
git clone https://github.com/shadmanAh/coursera
cd coursera
bundle
yarn
```

2. Run the migrations
```
rails db:create
rails db:migrate
```

3. Configure your development environment in config/environments/development.rb

4. Start the server
```
rails s
```
